Instructions — API Key

Open the Google Cloud Console: https://console.cloud.google.com/

Phase 1: Obtain your YouTube API Key
This is the only "manual" setup step required by Google.

Go to the Google Cloud Console.

Create a Project: Click the project dropdown in the top-left and select "New Project." Name it something like "YouTube-Assistant."

Enable the API: In the sidebar, go to APIs & Services > Library.

Search for "YouTube Data API v3".

Click it and hit Enable.

Create Credentials:

Go to APIs & Services > Credentials.

Click + Create Credentials at the top and select API Key.

Copy the key that appears. (You can click "Restrict Key" to limit it to just the YouTube API for better security).

Phase 2: Save the Code
Open your preferred text editor (VS Code, Sublime Text, or even Notepad).

Copy the entire block of HTML code from the project file.

Create a new file and paste the code inside.

Save the file to your desktop or a project folder as yt_tool.html.

Note: Ensure the file extension is .html and not .txt.

Phase 3: Launch and Configure
Open the file: Double-click yt_tool.html. It will open in your default web browser (Chrome, Firefox, or Edge).

Authenticate locally:

Find the Device Configuration box at the top.

Paste your API Key into the password field.

Click Save to Device.

Verification: You should see a status message saying "Key saved successfully." This key is now stored in your browser’s local storage and won't be visible in the source code.
